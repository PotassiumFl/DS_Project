## 编译说明

本项目使用 CMake 构建系统，建议使用 Ninja 作为生成器以提高构建速度。

### 环境要求
- CMake 4.0 或更高版本
- Ninja 1.8 或更高版本
- C++ 编译器（支持 C++17 标准）

### 编译步骤

1. **配置项目**
   ```bash
   cmake -G Ninja -S . -B .\build
   ```
   该命令将在 `build` 目录中生成构建文件。

2. **编译项目**
   ```bash
   ninja -C .\build
   ```
   编译后的可执行文件将位于 `build` 目录中。

### 快速编译（单行命令）
```bash
cmake -G Ninja -S . -B .\build && ninja -C .\build
```

## 使用说明

### 运行程序

程序需要通过命令行参数指定输入文件目录：

```bash
.\build\DS_Project.exe -P <文件目录路径>
```
或
```bash
.\build\DS_Project.exe --path <文件目录路径>
```

### 获取帮助信息

要查看程序的帮助信息，可以使用以下任一命令：

```bash
.\build\<可执行文件名> -H
```
或
```bash
.\build\<可执行文件名> --help
```

### 参数说明

| 参数 | 简写 | 说明 |
|------|------|------|
| `--path` | `-P` | 指定输入文件目录的路径 |
| `--help` | `-H` | 显示帮助信息 |

## 注意事项

1. 请确保指定的文件目录路径存在且包含有效的输入文件
2. 路径中如包含空格，建议使用引号括起来
3. 首次构建前，请确保已安装所需的依赖项
